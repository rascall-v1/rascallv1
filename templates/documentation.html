RASCALL Documentation

Overview

RASCALL (Rapid Approximate Spectral Calculations for ALL) is a tool for predicting and analyzing molecular spectra. It uses functional group theory to estimate the spectral features of molecules quickly and efficiently.

Features

List Function

The List function allows you to generate lists of molecules based on various criteria such as functional groups, molecular families, or specific atoms.

Usage:

./rascall_list [OPTIONS]

Options:
  --fg TEXT     List Molecules with Functional Group
  --mol TEXT    List Functional Groups in Molecule
  --mf TEXT     List Molecules in Family (choices: halo, hydro, all)
  --fw TEXT     Molecules in Frequency Window
  --ap TEXT     List Molecules with Specific Atoms
  --aw TEXT     List Molecules in Atmospheric Window (choices: co2, earth, methane)

Plot Function

The Plot function creates spectral plots for molecules or functional groups.

Usage:

./rascall_plot [OPTIONS]

Options:
  --fg TEXT     Plot Molecules with Functional Group
  --mol TEXT    Plot Single Molecule
  --mf TEXT     Plot Molecular Family (choices: halo, hydro, all)
  --fw TEXT     Molecules in Frequency Window
  --ap TEXT     Plot Molecules with Specific Atoms
  --aw TEXT     Plot Molecules in Atmospheric Window (choices: co2, earth, methane)

Search Function

The Search function allows you to look up detailed information about specific molecules using various identifiers such as SMILES, formula, name, etc.

Example Usage

List Example:

./rascall_list --fg C=O

This will list all molecules containing the C=O functional group.

Plot Example:

./rascall_plot --mol CH4 --mf hydro

This will plot the spectrum of methane (CH4) in the context of hydrocarbons.

Search Example:

Use the search box on the Search page to look up information about a specific molecule. For example, entering "CH4" will return detailed information about methane.
